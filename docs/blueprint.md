# **App Name**: Svensk Kassa

## Core Features:

- Valör Input: Möjliggör inmatning av antal för varje mynt- och sedelvalör.
- Delsummavisning: Visar delsumman per valör.
- Löpande Summor: Visar löpande summor för mynt och sedlar separat.
- Totalsumma: Visar den totala summan för alla valörer.
- Återställ Funktion: Möjlighet att återställa alla inmatningar.
- Export: Exportera data i olika format med datum, tid och kommentar.
- Import: Importera data i olika format med datum, tid och kommentar.

## Style Guidelines:

- Primary color: Light blue (#A0D2EB) to evoke a sense of calm and clarity, reminiscent of Swedish design principles, inspired by the clean Scandinavian aesthetic.
- Background color: Very light grey (#F5F5F5) to create a clean and touch-friendly light theme.
- Accent color: A slightly darker, but brighter blue (#64B5F6) to provide good contrast and highlight important interactive elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern look and a little warmth or personality; suitable for both headlines or body text
- Använd en ren och intuitiv layout som är touchvänlig.
- Använd tydliga och enkla ikoner för varje valör.
- Lägg till subtila animationer för att förstärka användarinteraktionen.
